LAB: Uso de la función print()

Escenario  
La función print() es una de las instrucciones más básicas de Python, ya que permite mostrar texto en pantalla.

Desarrollo del laboratorio  

En este ejercicio se realizaron las siguientes actividades:

Primero, se utilizó la función print() para mostrar el mensaje "¡Hola, Mundo!" en pantalla, usando comillas dobles.

Resultado:  
El programa se ejecutó correctamente mostrando el mensaje esperado.

Luego, se volvió a utilizar print(), pero esta vez para mostrar un nombre.

Resultado:  
El nombre se imprimió correctamente (ejemplo: print("Santiago")).

Después, se eliminaron las comillas dobles del texto y se ejecutó nuevamente el código.

Observación:  
Python generó un error indicando que el texto no está definido. Esto ocurre porque interpreta el contenido como una variable inexistente, produciendo un NameError.

A continuación, se eliminaron los paréntesis y se volvieron a colocar las comillas, ejecutando otra vez el programa.

Observación:  
Se produjo un error de sintaxis, ya que Python requiere que las instrucciones estén correctamente estructuradas, usando paréntesis o separadores adecuados.

Finalmente, se realizaron varias pruebas:

- Se cambiaron las comillas dobles por comillas simples  
- Se usaron varias funciones print() en una misma línea y en diferentes líneas  

Resultados:

- Las comillas simples funcionan igual que las dobles  
- Si se colocan varias funciones print() sin separación correcta, se genera un error  
- Si se separan correctamente (por ejemplo con coma), el código se ejecuta sin problema
